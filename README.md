# IT320-Software

## 📌 Project Overview

This project is a web-based application called **"Ainak"** that allows users to report city violations easily and quickly.
Users can submit reports with descriptions, images, and locations, while admins review and manage these reports.

---

##  Features

* User registration and login
* Submit violation reports
* View submitted reports
* Admin approval/rejection of reports
* Reward system for users

---

##  Technologies Used

* HTML
* CSS
* JavaScript
* (Future) Database integration

---

## 🗂️ Project Structure

* `Pages/` → HTML pages (home, login, signup)
* `CSS/` → stylesheets
* `Images/` → project images and logo
* `JavaScript/` → scripts

---

## 🚀 How to Run

1. Open the project folder
2. Open `home.html` in your browser
3. Navigate through the pages
